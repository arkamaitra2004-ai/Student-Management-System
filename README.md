# Student Management System

A console-based Student Management System developed in Java to manage student records efficiently. The application allows users to add, search, delete, and display student information through a menu-driven interface.

## Features
- Add Student
- Search Student by ID
- Delete Student by ID
- Display All Students

## Technologies Used
- Java
- ArrayList
- Object-Oriented Programming (OOP)

## Concepts Applied
- Classes and Objects
- Encapsulation
- Constructors
- Methods
- Collection Framework (ArrayList)
- CRUD Operations

## Future Enhancements
- Update Student Details
- File Handling for persistent storage
- MySQL database integration
- GUI using Java Swing or JavaFX
